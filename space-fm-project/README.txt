Hello! I am instructions for how to complete/maintain this website

Quick note: when adding images (or audio or any media), make sure to add them 
to add them to the appropriate folder or create one if there is not one already

1. The audio files for NW orbit still need to be implemented. To do this, create
an audio folder in this project and add the audio files you need. Then add them 
to the "src =" part of the audio tags in orbit.html. The current placeholder for 
these is "interview.mp3"

2. There is a blank link (src="#") for the "news and events" part of the 
carousel on the home page. Once there is a page for this, you can replace it 
with that.

3. When adding NW orbit artists, copy the format used for previous artists. That 
is, create a div with class of "thirdv2", then fill it with the appropriate 
title, description, image, and audio. When doing this, be sure to also update
the home page by replacing the image and description next to the carousel. This
is contained in the "container2" div.

4. When adding a new show, view the script.js document. Look for the appropriate
time, then create a conditional for that time and update the show, DJ, and image
path accordingly. View the other shows as examples.

5. There are links to download spotify and soundcloud playlists on the homepage.
Whenever there are files for this, be sure to add them.

Feel free to email me at wbowers@eastsideprep.org if you have any questions or 
find any lingering problems.

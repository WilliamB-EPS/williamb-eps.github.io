Project Case Study

Restaurant home page redesign

The Project

In web design 2, we had a project to do a homepage facelift, which means re-designing the homepage for a website online. Since I love food (and korean bbq), I decided to redesign the website for Palace KBBQ. This projects demonstrates both my skills as a web designer and vision as a graphic designer.

Fun facts

Despite being some of my best work in web design yet, this project was made only half way through web design 2! It is also responsive, meaning it works on computers, tablets, and phones, which has been a big step in my improvement as a web designer.

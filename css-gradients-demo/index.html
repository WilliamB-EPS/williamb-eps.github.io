CSS-Gradients Demo

By William

Welcome to my demo! In this demo, we will learn how to use CSS gradients to add style to our websites, and we will also learn some interesting applications of these gradients so you can get creative.

Step 0: Basic HTML & CSS

CSS gradients work with the background property, so be sure to create and size your div before beginning.

            
                <div class="gradient"> Welcome to my website! </div>

                .gradient-container {
                    width: 500px;
                    height: 300px;
                }

Step 1: Implement Your Gradient

If we think of CSS-gradients as a property of some div, then it is implemented like this.

            
                /* Linear gradient */

                .linear-gradient {
                    width: 300px;
                    height: 200px;
                    margin: 10px;
                    float: left;
                    background: linear-gradient(darkblue, purple);
                }

                /* Radial gradient */
                
                .radial-gradient {
                    width: 300px;
                    height: 200px;
                    margin: 10px;
                    float: left;
                    background: radial-gradient(darkblue, purple);
                }

                /* Conic gradient */

                .conic-gradient {
                    width: 300px;
                    height: 200px;
                    margin: 10px;
                    float: left;
                    background: conic-gradient(darkblue, purple);
                }

As you can see, linear gradients move in one direction, radial gradients move outwards from a circle, and conic gradients rotate around a point.

Step 2: Style Your Gradient

This step depends on which type of gradient you are using and how you want to style it. Below are some ways to style linear, radial, and conic gradients.

For linear gradients, we can change the direction of the gradient and add multiple colors.

    
            
                .linear-gradient-2 {
                    margin: 0 auto;
                    width: 500px;
                    height: 200px;
                    background: linear-gradient(45deg, darkblue, purple, darkred);
                }

For radial gradients, we can change the shape and the position of the center.

            
                .radial-gradient-2 {
                    margin: 0 auto;
                    width: 500px;
                    height: 200px;
                    background: radial-gradient(circle at top, darkblue, purple);
                }

For conic gradients, we can change the percentage of each color and add multiple colors.

            
                .conic-gradient-2 {
                    margin: 0 auto;
                    width: 500px;
                    height: 200px;
                    background: conic-gradient(darkblue 30%, purple 0 70%, darkred 0);
                }

Step 3 (optional): Get Creative!

One cool thing to do with linear gradients in make patterns. You can do this by utilizing the degree and transparency properties of the function. Here, we are also using set color percentages so that the gradient hard stops at those percentages (like we previously did for the conic gradient).

    
            
                .pattern {
                    margin: 0 auto;
                    width: 575px;
                    height: 200px;
                    background-color: purple;
                    background-image: linear-gradient(90deg, transparent 50%,rgba(255,255,255,.5) 50%);
                    background-size: 50px 50px;
                }

We can also make a border with gradients, and even combine that with the patterns we just talked about to make something cool!

    
            
                <div class="gradient-border"> 
                    <div> class="content-wrapper" Hello! </div> 
                </div>

                .gradient-border {
                    background-color: purple;
                    background-image: linear-gradient(90deg, transparent 50%,rgba(255,255,255,.5) 50%);
                    background-size: 50px 50px;
                    border-radius: 16px;
                    padding: 10px;
                }
                
                .content-wrapper {
                    background:#20283f;
                    font-size: 20px;
                    padding: 35px;
                    text-align: center;
                }
